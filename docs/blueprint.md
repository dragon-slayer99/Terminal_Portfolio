# **App Name**: Terminal Portfolio

## Core Features:

- Profile Display: Showcase personal details, contact info, and social links.
- Skills Showcase: Display coding proficiencies in an engaging terminal-like style.
- Project Gallery: Showcase development work with project details and external links.
- Experience Timeline: Detail professional journey and roles.
- Educational Background: Highlight academic achievements and institutions.
- Interactive Contact Form: Enables sending emails directly from the portfolio site.
- AI-Powered Personalization Tool: Suggests custom color palettes based on the user's activity.

## Style Guidelines:

- Primary color: Neon Cyan (#00FFFF) for a tech-inspired, futuristic look.
- Background color: Dark Slate (#2F4F4F) provides contrast and enhances readability.
- Accent color: Pure White (#FFFFFF) for clean accents and important text.
- Headline font: 'Orbitron' (sans-serif) for headings and titles, to give a futuristic, sci-fi terminal look. Note: currently only Google Fonts are supported.
- Body font: 'Roboto' (sans-serif) for readability and clean design in text paragraphs. Note: currently only Google Fonts are supported.
- Use a terminal-style interface with prompt-based navigation.
- Implement a vertical timeline layout for experience and education sections, mimicking Git logs.
- Use subtle animations for interactive elements, like text reveal on hover.
- Icons to match terminal aesthetic.
